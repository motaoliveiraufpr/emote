# BACRE-I_Emotion_Gaze_Mapping

The **BACRE-I Emotion Gaze Mapping module** extends the original battery by integrating 
eye-tracking data with emotional recognition tasks, allowing the analysis of:

- Visual exploration patterns
- Fixation distribution across facial regions
- Heatmap generation per stimulus
- Gaze-based attention metrics
- Correlation between gaze behavior and emotion recognition accuracy

This module enables deeper investigation of socio-cognitive processing 
through visual attention mapping.

---

## 🔬 Gaze Analysis Components

The system supports:

- Definition of Areas of Interest (AOIs):
  - Forehead
  - Eyes
  - Nose
  - Mouth
  - Lateral regions
  - Off-focus regions

- Fixation count per AOI
- Heatmap overlay generation
- Exportable visual analytics

---

## 📊 Generated Outputs

The Emotion Gaze Mapping module produces:

- Per-participant gaze distribution reports
- Aggregated fixation statistics
- Heatmap visualizations
- Correlation-ready datasets for statistical analysis
